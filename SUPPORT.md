# Support

## Getting help

If you need help using sn-sync:

1. Check project documentation under `docs/`.
2. Search existing issues to see if your question was already answered.
3. Open a new issue and include:
   - command used
   - expected behavior
   - actual behavior
   - relevant logs

## Bug reports and feature requests

- Bug reports: use the **Bug report** issue template.
- Feature ideas: use the **Feature request** issue template.

## Security issues

For security vulnerabilities, do not open a public issue.

Please follow `SECURITY.md`.
